# 오늘의 논문 — 2026-03-16

## SVG: 노이즈 제거 프레임 행렬을 통한 3D 입체 비디오 생성
*SVG: 3D Stereoscopic Video Generation via Denoising Frame Matrix*

**저자**: Peng Dai; Feitong Tan; Qiangeng Xu; David Futschik; Ruofei Du; Sean Fanello; Xiaojuan Qi; Yinda Zhang | **연도**: 2024 | **출처**: arxiv

### 추천 이유

스코어 기반 선정 (관련도: 0.71, 키워드: 0.00, LLM: 1.00)

### 요약

문제: 기존 단안 비디오 생성 모델은 발전했지만, 3D 스테레오스코픽 비디오 생성은 여전히 미개척 분야로 남아있다.
방법: SVG는 기존 단안 비디오 생성 모델을 활용, 추정된 깊이로 단안 비디오를 스테레오스코픽 뷰로 워핑한 후, 프레임 매트릭스 비디오 인페인팅 프레임워크와 disocclusion boundary re-injection scheme을 통해 pose-free, training-free 방식으로 3D 스테레오스코픽 비디오를 생성한다.
결과: 제안된 SVG는 scene optimization이나 model fine-tuning 없이 Sora, Lumiere 등 다양한 생성 모델에서 일관되고 의미론적으로 일관된 3D 스테레오스코픽 비디오를 생성하며, 이전 방법 대비 상당한 성능 향상을 보인다.
키워드: 3D Stereoscopic Video Generation, Denoising Frame Matrix, Video Inpainting, Monocular Video Generation, Disocclusion Boundary Re-injection

### 초록 (한국어)

비디오 생성 모델은 인상적인 단안 비디오(monocular video)를 생성하는 뛰어난 능력을 보여주었지만, 3D 입체 비디오(3D stereoscopic video)의 생성은 여전히 충분히 탐구되지 않았다. 우리는 상용 단안 비디오 생성 모델을 사용하여 3D 입체 비디오 생성을 위한 포즈 프리(pose-free) 및 학습 프리(training-free) 접근 방식을 제안한다. 우리의 방법은 추정된 비디오 깊이(video depth)를 사용하여 생성된 단안 비디오를 입체 기준선(stereoscopic baseline) 상의 카메라 뷰로 변형하고, 새로운 프레임 행렬 비디오 인페인팅(frame matrix video inpainting) 프레임워크를 활용한다. 이 프레임워크는 비디오 생성 모델을 활용하여 서로 다른 타임스탬프(timestamp)와 뷰(view)에서 관찰된 프레임을 인페인팅한다. 이 효과적인 접근 방식은 장면 최적화(scene optimization)나 모델 미세 조정(fine-tuning) 없이 일관되고 의미론적으로 일관된 입체 비디오를 생성한다. 더욱이, 우리는 잠재 공간(latent space)의 디스오클루전 영역(disoccluded areas)에서 전파되는 부정적인 영향을 완화하여 비디오 인페인팅의 품질을 더욱 향상시키는 디스오클루전 경계 재주입(disocclusion boundary re-injection) 기법을 개발한다. 우리는 Sora [4], Lumiere [2], WALT [8], Zeroscope [42]를 포함한 다양한 생성 모델의 비디오에 대한 실험을 수행하여 제안된 방법의 효능을 검증한다. 실험 결과는 우리의 방법이 이전 방법들에 비해 상당한 개선을 이룸을 보여준다. 코드는 \url{https://daipengwa.github.io/SVG_ProjectPage}에서 공개될 예정이다.
